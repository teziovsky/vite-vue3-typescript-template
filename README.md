## Get started 🏁

Clone the project and go to the project directory

```bash
git clone git@github.com:teziovsky/vite-vue3-typescript-template.git && cd vite-vue3-typescript-template
```

Install dependencies

```bash
npm install
```

Start the application!

```bash
npm run dev
```
